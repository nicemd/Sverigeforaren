Åkers (bimbo)

Kategori:Saknar allmän infoKategori:Saknar vägbeskrivning

Problem

  • Havoc
  • 6C+
  • Vid valvet. Start höger på givna grepp, sedan upp åt vänster, avsluta på juggen eller toppa ur upp i brötet om du vill.
  • Total Havoc Sans
  • 7B
  • Sittstart till Havoc, utan att använda arêten (inkl heelhook).
  • Vault Assault standup
  • 6B
  • Vid valvet. Start vänster vid stora greppen, catcha upp till kanten utan vänstra arêten.
  • Vault Assault
  • 7A
  • Sittstart med små krimpar, klättra upp och gör dynon utan vänsterkanten/arêten.
  • Limbo
  • 6C
  • Vid valvet. Sittstarta och klättra den vänstra arêten.
  • Before the Flood
  • 6C
  • På höga väggen runt hörnet från valvet. Sittstart mitt i överhänget och klättra rakt upp. Avsluta på juggen efter ca 4m. Resten borde kanske bultas.
  • Gods of the old World
  • 7A+
  • På höga väggen. Sittstarta och klättra den långa överhängande arêten. Avslutningen ej rensad, sluta på jug i sprickan tills vidare.
  • Bleau Vibrations
  • 7A
  • Uppe på platån. Sittstart till den överhängande arêten.
  • Bleau Vibrations Gauche
  • 7A
  • Sittstart som Bleau Vibrations, klättra halvvägs uppför arêten och vik sedan ut långt vänster till stor hylla.
  • Crazy Friday
  • 7A
  • Uppe på platån. Start höger med sidtag i sprickan och vänster hand på hyfsad krimp. Slap upp på rampen och sedan catch upp i klykan. Undvik jättegreppen till höger i sprickan.
  • Punch Out
  • 6B+
  • Går på platån. Starta med slopers på rampen (hoppa om du inte når), sedan dyno till kanten. Se till att inte flyga av när du får pendel, då kan det bli en ordentlig lufttur nerför höjden...
  • Beach Boyz
  • 6C
  • Går på första överhängsväggen. Sittstart vid arêten, utsteg höger.
  • Take me to the water
  • 6B+
  • Går på första överhängsblocket. Sittstart mitt på, rakt upp.
  • Tune in
  • 6B
  • Går på första överhängsblocket. Sittstart till höger mot diedret, klättra rakt upp med två kanter. Stå inte på stenen till höger.
  • Talus
  • 4A
  • Uppe i klyftan. Start med händerna i tvärsprickan, sedan upp höger.
  • Ur askan
  • 5A
  • Sittstart vänster om Bleau Vibrations uppe vid platån.
  • Trollsländan
  • 7A
  • På vänstra delen, nästan längst till vänster. Startar med HH på undercut, VH på stor sloper.
  • Pocketparty
  • 5B
  • Vägg ca 200 m från Trollsländan i riktning mot Åkers. Problemet går strax vä om aréten.
  • The wild one
  • 6B
  • Aréten höger om Pocketparty. Stora rampen i starten ingår ej.
  • Soft vibrations
  • 7A+
  • Som Bleau Vibrations, men avsluta med Punch out.
  • Total Hammock
  • 7A
  • Sittstarten under Havoc. Allt är tillåtet; heelhook, grepp på arêten, you name it.
  • Födelsedagskalas
  • 7A
  • Överhängade väggen längst vänster på stora väggen. Direktlinje via 3 horisontella sprickor.
  • Blod o tårar
  • 6C+
  • Väggen vid krypin-grottan. Högra aréten. Startar med hh i sprickan, vh valfritt.
  • Return of the caveman
  • 7A
  • Väggen vid krypin-grottan. Högra aréten. Startar vid boxen i två undercling. Catcha med vh upp på boxen, följ sedan aréten uppåt.
  • Jordfräs
  • 5C
  • Höger om entrén till muren. Start strax höger om vä-aréten, följ stora sidgrepp upp, avsluta åt vänster (escape)
  • Brand new demons
  • 6A+
  • Sprickan ca 2 m vänster om Gods of the old world (stå-start 6a).
  • Wild Thing
  • 7C
  • Havocväggen. Sittstart mitt på, gå upp till två snedställda grepp och gör en monkeyslap upp till rampen i slutet av Havoc. Den bästa och mest direkta linjen uppför väggen! Det är möjligt att man kan klättra in i Havoc men det lär snarare bli hårdare.
  • Sabretooth
  • 7B
  • Den perfekta överhängade arêten samma vägg som Return of the caveman. Start från botten, krama arêten via två sidtag och forcera sedan upp åt höger.
  • Return of the Caveman assis
  • 7A+
  • Sittstart till Return of the caveman. Staretar från en bra sloper med vänster och en incutkrimp i en spricka.
  • Vault Assault au naturell
  • 6C
  • Start som Vault assault assis, men utan uteslutning av stora greppen på areten efter första moven. Naturligaste linjen från den sittstarten.
  • Wild Thing standup
  • 7A+
  • Stand-up varianten till Wild Thing, start vanster om Havoc, skicka langt upp hoger till det stora greppet med hoger hand.
  • Fredriks bildningsluckor
  • 6B+
  • Start i sprickan till höger om Födelsedagskalas. Sittstart till denna är gjord av J. Smedsaas 7a?
  • Neanderthal
  • 6A+
  • Vägg hö om Caveman. Start på jugge nära granen. Går svagt till vä.
  • Camouflage
  • 7A
  • Precis hö om Brand new demons(som är eliminerad). Studs till jugg med hö h, vä h upp till sidocrimp. Långt sträck åt hö o toppa ur.
  • Belle de jour
  • 6A+
  • Vägg hö om vault. 2 m till hö om sprickan (eliminerad). Start match vä h sidotag, hö h list, gå rakt upp via slopande list t vä.
  • Vinglig
  • 6B+
  • Vägg till hö om vault. Innanför träd. Starta brett med vä h sidotag hö h sidotag. Klättra rakt upp.
  • Kunglig
  • 6C
  • Vägg hö om vault. Hö om träd. Start vä h sidotag studsa till list med hö h.
  • Köttjuven
  • 7B
  • Blocket till vänster innan Gods... Sittstart på vänstra delen och klättra rakt upp över svaskölden. Arêten är off om den nu ens går att använda.
  • Real assis
  • 7A
  • Längst till vänster på samma vägg som Gods... Sittstarta, dra upp till juggarna och vidare till ett bra grepp på läppen.
  • She's a fox
  • 6C+
  • Det lutande blocket vid return of the caveman. Sittstart från botten. Antagligen morpho.
  • Facelift
  • 7B
  • Sittstart i överhänget höger om Gods... Klättra upp höger på de slopiga hyllorna runt arêten. Ca 20 moves och ett mentalt crux ovanför en jobbig landing.
  • ÅKERS FÖRARE
  • -
  • Borderline assis
  • 7C+
  • Sittstart som Bleau vibrations, sträck höger till krimpar och klättra rakt upp utan att använda arêten.
  • The Turd
  • 6C+
  • Runt hörnet till höger om Havoc. Start i stora sidtaget (höger hand åtminstone). snett vänster sen rakt upp via stora grepp i sprickan.
  • Weissmüller
  • 7A
  • Hoppstart ca 1m höger om trollsländan, sedan campus upp åt vänster (eller använd fötterna om du vill). Slutar typ som trollsländan.
