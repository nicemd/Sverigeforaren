  • Enigma
  • 8A+
  • Den uppenbara linjen rakt framför parkeringen. Går ej att missa. Brant överhäng med hård start.
  • Megagrip
  • 7C+
  • Arêten vä om Enigma, start på ett undercling.
  • Frasradikal
  • 6C+
  • Ståstart till höger om Minigrip. Baksidan av Enigma. SS: lätt 7a.
  • Borgarbrackor
  • 6C
  • Till höger om Frasrafikal strax vänster om areten.
  • Minigrip
  • 7B+
  • Mitt på baksidan av Enigmablocket, start med två tiny crimpsidtag och skicka ut vä till bra grepp, sen upp till jug.
  • Bergakungen
  • 7B
  • Överhäng på baksida om sten tv om enigmablocket. Start typ SS med VH på örat. Sedan rakt upp svagt åt höger.
